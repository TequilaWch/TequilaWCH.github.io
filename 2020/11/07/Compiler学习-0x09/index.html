2：算符优先分析&错误处理

给出(id+*id进行算符优先分析（包括错误恢复）的完整过程

先构造如下的算符优先表

id + * ( ) $
id ⋗ ⋗ ⋗ ⋗
+ ⋖ ⋗ ⋖ ⋖ ⋗ ⋗
* ⋖ ⋗ ⋗ ⋖ ⋗ ⋗
( ⋖ ⋖ ⋖ ⋖ ≐
) ⋗ ⋗ ⋗ ⋗
$ ⋖ ⋖ ⋖ ⋖

再构造如下的错误恢复表和有限错误类型

id ( ) $
id e3 e3 ⋗ ⋗
( ⋖ ⋖ ≐ e4
) e3 e3 ⋗ ⋗
$ ⋖ ⋖ e2 e1

1.e1：插入id到输入缓冲，“缺少运算数”

2.e2：删除)，“未匹配右括号”

3.e3：插入+到输入缓冲，“缺少运算符”

4.e4：弹出(，“缺少右括号”

5.e5：+或*两侧不全有非终结符时，忽略此符号，错误信息“缺少运算数”

根据上表进行算符优先分析和错误处理的过程如下(因为不会打这几个符号且没有涉及，用<>=代替)

$( id+*id$ ( < id
$(id +*id$ id > +
$(E +*id$ ( < +
$(E+ *id$ + < *
$(E+* id$ * < id
$(E+*id $ id > $
$(E+*E $ * > $ : e5
$(E+E $ + > $
$(E $ e4
